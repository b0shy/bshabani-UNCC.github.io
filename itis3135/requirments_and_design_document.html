Flavors of Palestine

Bashar Shabani

10/15/2021

Project Name: Flavors of Palestine

Project Overview:

  • The application I am creating is a website that will contain Palestinian recipes passed from my mothers’ family. The purpose of this is to introduce people to the Palestinian culture through its cuisine.
  • The intended users of this website are people seeking to experience a new and different culture through its cuisine.
  • The website will contain content varying from breakfast, lunch, and dinner items. This will include things from cold to hot food, salads, appetizers, main dishes, drinks, and dessert.

Client Information:

  • The client’s name is Lana Obeid
  • The client is associated with a local group called Palestinian Cuisine
  • The clients email address: lanaobeid7@hotmail.com

Site Map:

Page Design [Home Page]:

  • Page Name: Flavors of Palestine
  • Purpose: The home page will contain a navigation bar to allow users to go from page to page. It will also contain a photo gallery displaying images of Palestinian dishes, as well as an about section talking about the page and what is has to offer and a comments section where users can leave input.
  • Audience: The audience of this page will be people seeking to experience a new and different culture through its cuisine.
  • Content:
    • This is a page where users can enter data.
    • The data fields will not require any validation
    • This page will contain a drop down menu, which will contain hyperlinks to the other pages of the website.
    • The actions that will happen on this page are navigation to other pages and user data entry.

Page Design [Breakfast]:

  • Page Name: Breakfast
  • Purpose: The breakfast page will contain numerous Palestinian breakfast recipes. Each dish will have the recipe, an image of the dish, and a short description of the dish
  • Audience: People looking for Palestinian breakfast dishes and their recipes.
  • Content:
    • This is not a page where users can enter data.
    • This page will contain hyperlinks to take you to the individual recipe
    • The actions that will happen on this page are navigation to individual recipes.

Dynamic Functionality on the Website:

  • I will implement a comments section where the user will fill out a form with two text boxes, one for the users name and one for whatever comments the user has. This will be implemented on the home page because it allows the user go back to the start page and leave a comment, rather than have the user go searching through the navigation bar for the page that contains the comments section.
  • I will implement a photo gallery. This photo gallery will contain photos of all of the recipes mentioned on the website. This will be implemented on the home page because it entices the user to go and look for the recipe of the visually appealing dish.
